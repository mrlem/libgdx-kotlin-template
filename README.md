# libgdx-kotlin-template

A modern project template featuring:
* gradle build:
    - gradle 8.8
    - build logic & convention plugins
    - TOML versions file
* kotlin 1.9.22
* libgdx 1.12.1
    - KTX extensions: for more idiomatic kotlin syntax
    - GLTF: for high quality 3D rendering
    - screen-based base application
